My Projects

MedicoBot: A Disease Predictor

MedicoBot is an intelligent medical assistant designed to predict potential diseases based on patient-reported symptoms. This system leverages an Ensemble Learning model to provide highly accurate diagnoses, achieving an impressive 99% accuracy. Beyond prediction, it offers actionable recommendations such as precautionary measures and appropriate exercises for better health management.

Band-o-bast: Remote Tracking System for Security Agents

Band-o-bast is an IoT-powered wearable device aimed at enhancing the operational efficiency of security agents. It tracks real-time locations of on-ground personnel, enabling remote monitoring through a web application. This project was showcased at the Grand Finale of Smart India Hackathon 2023, demonstrating its potential for large-scale deployment in security and defense sectors. The device uses advanced networking technologies like LoRa for reliable and long-range communication, ensuring seamless connectivity even in challenging environments.

Well-Being Arcade: Mini Game for Mental Health Assessment

Well-Being Arcade is a web-based platform designed to assess students' mental health through a gamified approach. It integrates a series of engaging questionnaires that assess various aspects of mental well-being in a non-intrusive manner. The front-end of the application is built using HTML, CSS, Bootstrap, and JavaScript, creating an intuitive and user-friendly interface aimed at encouraging regular mental health check-ins among students.

Sentimental Analysis of Restaurant Review

This project involves the development of a machine learning model that performs sentiment analysis on customer reviews from restaurants. By leveraging Natural Language Processing (NLP) and various machine learning algorithms, the system analyzes reviews to determine customer sentiment—whether positive, negative, or neutral. This project was part of the IBM SkillsBuild program, where I also gained proficiency in Python and its libraries to develop and train the model.

Interactive Website for a Shop

I developed an interactive website for a shop that serves both the shopkeeper and customers. The website allows customers to browse available products and make orders online, while also providing the shopkeeper with tools for inventory management. The project involved creating a responsive and user-friendly interface, ensuring smooth functionality across different devices and offering a seamless shopping experience.
